Image Manipulation

I created a series of image manipulation tools using the HTML5 canvas tag and some JavaScript workers. I was able to perform a multitude of tasks including: pixel sorting, pallete limiting with dithering, and k-means blotching (as seen above).

My favorite part of this project was the Fourier epicycle generator that can be seen on the home page. It can be given any array of points (typically generated from an SVG file) and it will create the circles necessary to follow that path.

I hope to soon have a user-friendly web-app available to play around with all of the tools, but until then, I've included some images that I generated as well as a link to a GitHub repository.
